# Lidl++ (Lidl Plus Plus)

## Story:  
The project we made in a hackathon-like course about AI.  
We did not use AI in the making of this project (only for frontend), because Razvan.  

## Project Idea:  
We made an AI model that makes a store configuration based on some products it gets as input on predefined **2D** store maps.  
We have an UI where the store managers and/or staff type all the products they have and we will return a map with all the items in the optimal order.  

## Technology
`Python Flask` as a backend server and `React.js` as a frontend UI.
The AI model was implemented in `Python` as well.
